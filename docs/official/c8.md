# c8 Coverage Documentation

**Repository Version:** ^9.0.0  
**Official Documentation:** https://github.com/bcoe/c8  
**Latest Release:** April 2026

## Overview

c8 is a code coverage tool for Node.js that uses V8's built-in coverage functionality. It provides fast, accurate coverage reporting with minimal overhead, making it ideal for modern JavaScript and TypeScript projects.

In this monorepo, c8 serves as the primary code coverage solution, integrated with Vitest and Turbo to provide comprehensive coverage analysis across all packages and applications.

## Key Features

### **V8 Native Coverage**
- **Built-in Coverage**: Uses V8's native coverage collection
- **Zero Configuration**: Works out of the box with minimal setup
- **Fast Performance**: Minimal overhead during test execution
- **Accurate Reporting**: Precise line, branch, and function coverage

### **Reporting Options**
- **Multiple Formats**: HTML, JSON, LCOV, text, and console reports
- **Detailed Reports**: Per-file and per-function coverage details
- **Threshold Enforcement**: Configurable coverage thresholds
- **Exclusion Support**: Flexible file and directory exclusion patterns

### **Integration Support**
- **Test Runner Integration**: Works with Vitest, Jest, and other test runners
- **CI/CD Ready**: Designed for continuous integration workflows
- **Monorepo Support**: Optimized for monorepo structures
- **TypeScript Support**: Full TypeScript coverage reporting

## Installation

### Core Dependencies

```bash
# Install c8
pnpm add -D c8@^9.0.0

# Install with Vitest (recommended)
pnpm add -D vitest@^3.0.0 @vitest/coverage-v8@^3.0.0
```

## Configuration

### **c8 Configuration**

```javascript
// c8.config.js
export default {
  // Coverage collection options
  reporter: ['text', 'json', 'html'],
  reportsDir: './coverage',
  exclude: [
    'coverage/**',
    'dist/**',
    '.next/**',
    '.turbo/**',
    '**/*.config.{js,ts,mjs}',
    '**/*.d.ts',
    '**/test/**',
    '**/__tests__/**',
    '**/*.test.{js,ts,jsx,tsx}',
    '**/*.spec.{js,ts,jsx,tsx}',
    'packages/testing/**', // Exclude test utilities
    'node_modules/**'
  ],
  
  // Coverage thresholds
  thresholds: {
    global: {
      branches: 80,
      functions: 80,
      lines: 80,
      statements: 80
    }
  },
  
  // Additional options
  all: true, // Include all files, not just those with tests
  allowExternal: false, // Don't collect coverage from external files
  checkCoverage: true, // Fail if thresholds are not met
  clean: true, // Remove coverage results before running
  dryRun: false, // Don't write coverage files
  omitRelative: false, // Show relative paths in reports
  src: ['src/**'], // Source files to include
  
  // Performance options
  cache: true, // Enable caching
  excludeAfterRemap: false, // Exclude files after remapping
  include: ['src/**'], // Files to include in coverage
  
  // Reporting options
  reporterOptions: {
    html: {
      subdir: 'html'
    },
    json: {
      file: 'coverage.json'
    }
  }
};
```

## Usage

### **Basic Coverage**

```bash
# Run tests with coverage
pnpm test:coverage

# Generate HTML report
pnpm test:coverage:html

# Generate JSON report
pnpm test:coverage:json

# Check coverage without running tests
pnpm coverage:check
```

### **Monorepo Coverage**

```bash
# Run coverage for all packages
pnpm turbo run test:coverage

# Run coverage for specific package
pnpm turbo run test:coverage --filter=@agency/ui-components

# Run coverage for affected packages
pnpm turbo run test:coverage --filter="*"
```

### **Coverage Reports**

```bash
# View console report
pnpm test:coverage

# Open HTML report
open coverage/index.html

# View JSON report
cat coverage/coverage.json | jq .
```

## Integration Patterns

### **Turbo Integration**

```json
// turbo.json
{
  "tasks": {
    "test": {
      "dependsOn": ["build"],
      "outputs": ["coverage/**"],
      "inputs": ["src/**", "test/**", "package.json", "vitest.config.*"],
      "cache": true
    },
    "test:coverage": {
      "dependsOn": ["build"],
      "outputs": ["coverage/**"],
      "inputs": ["src/**", "test/**", "package.json", "vitest.config.*"],
      "cache": true
    }
  }
}
```

### **GitHub Actions**

```yaml
# .github/workflows/coverage.yml
name: Coverage

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  coverage:
    runs-on: ubuntu-latest
    
    steps:
      - uses: actions/checkout@v4
      
      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: '22'
          cache: 'pnpm'
      
      - name: Install dependencies
        run: pnpm install --frozen-lockfile
      
      - name: Run tests with coverage
        run: pnpm turbo run test:coverage
      
      - name: Upload coverage to Codecov
        uses: codecov/codecov-action@v3
        with:
          file: ./coverage/lcov.info
          flags: unittests
          name: codecov-umbrella
```

### **Coverage Comments**

```yaml
# .github/workflows/coverage-comment.yml
name: Coverage Comment

on:
  pull_request:
    branches: [main]

jobs:
  coverage-comment:
    runs-on: ubuntu-latest
    
    steps:
      - uses: actions/checkout@v4
      
      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: '22'
          cache: 'pnpm'
      
      - name: Install dependencies
        run: pnpm install --frozen-lockfile
      
      - name: Run tests with coverage
        run: pnpm turbo run test:coverage
      
      - name: Coverage Comment
        uses: actions/github-script@v7
        with:
          script: |
            const fs = require('fs');
            const coverage = JSON.parse(fs.readFileSync('./coverage/coverage-summary.json', 'utf8'));
            
            const comment = `
            ## Coverage Report
            
            | Metric | Coverage |
            |--------|----------|
            | Lines | ${coverage.total.lines.pct}% |
            | Functions | ${coverage.total.functions.pct}% |
            | Branches | ${coverage.total.branches.pct}% |
            | Statements | ${coverage.total.statements.pct}% |
            
            [View detailed report](https://github.com/${{ github.repository }}/actions/runs/${{ github.run_id }})
            `;
            
            github.rest.issues.createComment({
              issue_number: context.issue.number,
              owner: context.repo.owner,
              repo: context.repo.repo,
              body: comment
            });
```

## Troubleshooting

### **Common Issues**

#### **Coverage Not Generated**

```bash
# Problem: No coverage files generated
# Solution: Check V8 coverage support
node --version  # Ensure Node.js supports coverage

# Check if coverage is enabled
pnpm test:coverage --debug
```

#### **Missing Coverage**

```bash
# Problem: Low coverage despite tests
# Solution: Check file inclusion/exclusion
c8 report --include='src/**' --exclude='test/**'

# Check if tests are actually running
pnpm test --run
```

#### **Threshold Failures**

```bash
# Problem: Coverage thresholds not met
# Solution: Check specific files
c8 check --verbose

# Generate detailed report
c8 report --reporter=html
```

### **Debugging Tools**

```bash
# Debug coverage collection
DEBUG=c8:* pnpm test:coverage

# Check coverage for specific files
c8 report --include='src/components/**'

# Verify configuration
c8 check --config c8.config.js
```
